LexiDecay — instant, explainable text classification

A blazing-fast, semi-supervised lexical decay classifier that scores and classifies text without training. Adaptive frequency decay, IDF toggles, and transparent scoring make LexiDecay ideal for topic classification, intent detection and lightweight cognitive systems.

Python Quickstart

# Install
pip install LexiDecay

# Basic usage
from LexiDecay import LexiDecayModel
m = LexiDecayModel()
m.add_category("science", open("science.txt").read())
m.add_category("philosophy", open("philosophy.txt").read())
text = "Quantum theories explore the probabilistic structure of the universe."
result = m.classify(text)
print(result["top"])  # ('science', score, probability)
